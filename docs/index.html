الموظفون والأقسام

متابعة حالة الدوام، طلبات الاستراحة، والتقارير التشغيلية المرتبطة بها.

الإجازات والموافقات

تجربة التطبيق

شاشات حقيقية من Mawared ERP

واجهة Android عربية مصممة لمسارات العمل اليومية. الصور تستخدم بيانات تجريبية أو بيئة مرحلية مضبوطة.

لا تحتوي الصور العامة على كلمات مرور أو مفاتيح API أو بيانات عملاء حقيقية.

المعمارية العامة

فصل واضح بين التطبيق، الأعمال، والبيانات

تطبيق Android يتصل بخادم خلفي مسؤول عن المصادقة والصلاحيات وقواعد الأعمال وعزل بيانات الشركات.

  • واجهة Android باستخدام Kotlin وJetpack Compose
  • خادم TypeScript وNode.js وExpress
  • Prisma مع PostgreSQL على Supabase
  • JWT access وrefresh-session architecture
  • عزل بيانات الشركات وفرض الصلاحيات من الخادم
