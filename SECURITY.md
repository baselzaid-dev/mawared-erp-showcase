# Security Policy

## Scope

This repository is a public product showcase and does not contain the commercial Mawared ERP source code, production credentials, private API configuration, or customer data.

Security reports may still be relevant when they concern:

- Accidentally published secrets or personal information.
- Unsafe links or downloadable files added to this showcase.
- Public documentation that unintentionally exposes sensitive operational details.
- A verifiable security issue affecting an official Mawared deployment.

## Reporting

Please do not publish sensitive technical details, credentials, personal data, or exploit instructions in a public issue.

Contact the repository owner privately through the contact options available on the owner's GitHub profile. Include:

- A clear description of the issue.
- The affected page, component, or deployment.
- Reproduction steps that avoid accessing or modifying other users' data.
- The potential impact.
- Any safe evidence needed to verify the report.

## Responsible testing

Do not perform destructive testing, credential attacks, denial-of-service testing, automated high-volume scanning, social engineering, or access to data that does not belong to you.

Submitting a report does not grant permission to test private systems or bypass access controls.
